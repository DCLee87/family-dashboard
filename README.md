# Family Dashboard

가족 구성원이 일정, 할 일, 공지, 날씨와 위치 상태를 안전하게 공유하는 가정용 대시보드입니다.

## 현재 단계

- 요구사항 분석(Inception)
- 구현 및 기술 스택은 미정
- 시놀로지 2베이 NAS를 운영 서버로 사용하는 것을 목표로 함

## 문서 작성 원칙

- 합의된 사항과 미결정 사항을 구분한다.
- 주소, 실제 위치, 비밀번호, PIN 등의 비밀값은 저장소에 기록하지 않는다.
- 중요한 설계 결정은 ADR로 이유와 함께 남긴다.
