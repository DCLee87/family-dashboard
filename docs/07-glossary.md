# 용어집

| 용어 | 정의 |
|---|---|
| 관리자 | 계정, 권한, 기기 화면, 보안 및 시스템 설정을 관리하는 부모 사용자 |
| 일부 편집 | 일정 추가·수정, 할 일 추가·수정·완료, 공지·메모 작성 권한 |
| 보기 전용 | 정보를 조회할 수 있으나 변경할 수 없는 상태 |
| 공용 편집 PIN | 가족 공용 태블릿에서 일부 편집 기능을 잠금 해제하는 공통 비밀번호 |
| 기기 프로필 | TV, PC, 태블릿, 모바일별 표시 항목과 배치 정책 |
| 위젯 | 일정, 할 일, 날씨 등 대시보드를 구성하는 독립 표시 단위 |
| 일정 기반 상태 | 일정의 대상 가족, 장소 및 시간으로 계산한 예상 상태. 실제 GPS 위치가 아님 |
| UP | 위험 중심의 반복적·점진적 개발 프로세스인 Unified Process |
| ADR | 중요한 아키텍처 또는 설계 결정과 이유를 기록한 문서 |
| 인수 기준 | 요구사항이 충족되었다고 판단하기 위한 관찰·측정 가능한 조건 |
| 시스템 기본 시간대 | 일정과 알림 계산에 사용하는 가족 시스템의 기준 시간대. 초기값은 `Asia/Seoul` |
