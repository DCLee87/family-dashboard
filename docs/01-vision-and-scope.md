# 비전과 범위

## 비전

가족 구성원이 일정, 할 일, 공지, 날씨와 필요한 경우 위치 상태를 안전하게 공유하고, PC·TV·모바일·태블릿에서 확인하거나 편집할 수 있는 가정용 대시보드를 시놀로지 NAS에 구축한다.

## 목표

- 가족이 필요한 생활 정보를 한 화면에서 확인한다.
- 기기의 크기와 사용 목적에 맞는 정보를 제공한다.
- 부모는 쉽게 관리하고 딸은 안전하게 열람한다.
- 가족 데이터는 가정의 시놀로지 NAS에서 운영한다.

## 초기 범위

- 사용자: 본인, 배우자, 딸 1명
- 기기: PC, 거실 TV, 가족 공용 태블릿, 부모 모바일
- 정보: 일정, 할 일, 공지·메모, 날씨, 위치 또는 상태

## 초기 범위에서 제외

- 친척이나 외부 사용자 초대
- 딸의 전용 모바일 앱
- 구체적인 기술 스택과 배포 구조(추후 결정)
- 월간, 매월 N번째 요일 등 복잡한 반복 일정 규칙

## 미결정 사항

- 삭제 및 복구 정책
- 실시간 GPS 위치 기능의 후속 도입 여부
- 외부 캘린더 연동
- 알림 방식
- NAS 외부 접속 및 장애 대응

## 위치 기능 범위

- 초기 버전은 실시간 GPS를 수집하지 않는다.
- 등록된 일정의 장소와 시간으로 가족의 예상 상태를 표시한다.
- 예상 상태는 실제 현재 위치로 표현하지 않는다.
