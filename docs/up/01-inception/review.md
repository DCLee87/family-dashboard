# Inception 종료 검토

- 검토일: 2026-07-13
- 결과: Elaboration 진행 승인

## 비전

가족 구성원이 일정, 할 일, 공지, 날씨와 일정 기반 예상 상태를 안전하게 공유하고 PC·TV·iOS·Android·태블릿에서 확인·관리할 수 있는 PWA를 DS216+II에서 운영한다.

## 초기 버전 우선순위

### Must

- 기기별 가족 대시보드
- PIN 기반 관리자 모드
- 일정, 반복, 공개 범위와 일정 기반 예상 상태
- 할 일, 완료와 반복 회차
- 공지·메모 게시 항목
- 기기 등록·폐기와 안전한 외부 접속
- 권한에 따른 서버 응답 필터링

### Should

- 일정·할 일 Web Push
- 장소 기반 날씨
- 휴지통과 복원
- 중요 공지 Push
- 기기별 위젯 구성
- 겹치는 일정 경고
- PIN 복구

### Could

- 공용 태블릿 Push
- 상세 완료 이력 관리
- 고급 날씨 화면 설정

## 제외 및 후속 후보

- 실시간 GPS
- 네이티브 iOS·Android 앱과 앱스토어 배포
- 삼성 TV 전용 `.wgt` 앱
- 카카오톡·문자·이메일 알림
- 복잡한 월간 반복
- 사용자별 공지 읽음 추적
- 외부 친척 계정

PWA가 실제 부모 기기에서 필수 품질 기준을 충족하지 못하면 모바일 앱을 후속 대안으로 재평가한다.

## 주요 위험

1. DS216+II의 1 GB 메모리
2. 외부 접속의 가족 데이터 노출
3. 삼성 TV 브라우저의 장시간 호환성
4. iOS·Android PWA와 Web Push 실기기 동작
5. 외부 백업 부재

백업 정책은 후속 검토로 연기했지만 실제 가족 데이터 운영 전에는 반드시 해결한다.

## 품질 기준

ISO/IEC 25010:2023 제품 품질 모델을 참조하며 성능 효율성, 호환성, 상호작용성, 신뢰성, 보안성, 유지보수성, 유연성과 안전성 요구사항을 `docs/05-quality-requirements.md`에서 추적한다.

## 단계 전환 결정

- 범위와 사용자 가치가 충분히 명확하다.
- 핵심 위험은 식별됐으며 Elaboration 반복에서 검증할 수 있다.
- 구현을 바로 확대하지 않고 실행 가능한 아키텍처 골격과 실기기 시험을 먼저 수행한다.
- 따라서 Inception을 종료하고 Elaboration E1을 시작한다.
