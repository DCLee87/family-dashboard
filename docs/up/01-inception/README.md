## 현재 주요 결정

- 본인과 배우자는 관리자이며 딸은 보기 전용이다.
- PC·부모 모바일·공용 태블릿은 PIN으로 관리자 모드를 잠금 해제하며 TV는 항상 보기 전용이다.
- PIN 분실 복구는 등록된 PC에서 복구 코드로만 수행한다.
- TV·PC·태블릿·모바일은 적응형 기기 프로필을 사용한다.
- 초기 위치 기능은 GPS가 아니라 일정 기반 예상 상태를 사용한다.
- 일정에는 대상 가족을 1명 이상 지정하며 여러 명을 함께 선택할 수 있다.
- 반복 일정은 매주 복수 요일과 특정 회차 예외 처리를 지원한다.
- 겹치는 일정은 경고 후 허용하며 예상 상태는 시작 시각이 가장 늦은 일정을 우선한다.
- 일정은 가족 공개, TV 요약, 부모 전용 공개 범위를 지원한다.
- 새 일정의 기본 공개 범위는 가족 공개다.
- 부모 전용 일정은 권한 없는 화면에서 `개인 일정 · 종료 시각`으로 비식별 표시한다.
- 삭제 일정은 30일간 휴지통에 보관하며 삭제·복원·영구 삭제는 관리자 모드에서 수행한다.
- 종일·여러 날 일정을 지원하며 별도 상태 시간대가 없으면 예상 위치에 반영하지 않는다.
- PC·TV·iOS·Android는 하나의 적응형 PWA를 사용하고 NAS의 단일 경량 컨테이너에서 제공한다.
- 부모 모바일은 Web Push를 지원하고 공용 태블릿은 선택, TV는 화면 강조를 사용한다.
- 부모 일정은 해당 부모가, 딸·부모 공동·가족 일정은 부모 두 명이 기본 알림 수신자가 된다.
- 시간 일정은 30분 전, 종일 일정은 전날 오후 8시 알림을 기본으로 한다.
- 할 일은 담당 가족 1명 이상, 선택적 마감·메모, 보통·중요 우선순위와 완료 상태를 가진다.
- 반복 할 일은 매일 또는 매주 선택 요일을 지원하고 회차마다 독립적인 완료 상태를 가진다.
- 마감 시각이 있는 할 일은 1시간 전, 마감일만 있으면 당일 오전 9시에 기본 알림을 보낸다.
- 기한 지난 할 일은 화면에서 강조하고 `중요` 항목만 매일 오전 9시에 재알림한다.
- 완료 할 일은 7일 후 기본 화면에서 숨기고 최소 1년 기록하며 직접 삭제한 할 일은 30일간 휴지통에 보관한다.
- 공지와 메모는 하나의 게시 항목으로 관리하며 종류·중요도·기간·공개 범위로 표시 방식을 결정한다.
- 중요 공지는 부모 모바일에 Push하고 공용 화면 상단에 고정하며 사용자별 읽음 추적은 초기 범위에서 제외한다.
- 등록 장소를 시·군·구 수준의 날씨 지역과 연결하고 집 및 현재 허용된 일정 지역의 날씨를 표시한다.
- 날씨는 현재·24시간·7일 데이터를 공통 수집하고 기기별 기본 표시 범위를 다르게 구성한다.
- 현재·시간별 날씨는 30분, 일별 예보는 3시간마다 갱신하고 6시간·24시간 실패 상태를 구분한다.
- 등록된 부모 기기는 외부에서 HTTPS로 조회하고 PIN으로 관리자 모드를 잠금 해제할 수 있다.
- 부모 모바일은 집 안에서 PC가 발급한 10분 일회용 QR 코드와 PC 최종 승인으로 등록한다.
- TV는 6자리 코드로 로컬 보기 전용 등록하고 공용 태블릿은 PC의 QR 승인으로 로컬 공용 기기 등록한다.
- 거실 TV는 Samsung KQ85QC80AFXKR 내장 브라우저를 1차 대상으로 하고 실제 호환성 시험을 수행한다.
- TV 가독성은 약 3 m 시청 거리를 기준으로 실제 기기에서 검증한다.
- TV는 브라우저 즐겨찾기로 수동 시작하며 Tizen Web `.wgt` 앱은 후속 후보로 둔다.
- TV는 일정·할 일 변경을 60초, 중요 공지를 10초 이내 반영하고 연결 실패 시 30초 HTTP 갱신으로 전환한다.
- 컨테이너 최초 실행 코드로 집 안 PC를 최초 신뢰 PC로 등록하고 PIN·복구 코드를 설정한다.
- 등록 기기는 수동 폐기 전까지 유지하고 90일 미사용 시 경고하며 인증정보는 자동 갱신한다.
- 시놀로지 2베이 NAS를 운영 서버 후보로 사용한다.

## 미결정 항목

- 공지와 메모의 구분
- 날씨 공급자
- DS216+II의 실제 여유 메모리와 디스크 구성(Container Manager, DSM 및 모델 공식 사양은 확인됨)
- RAID 유형 최종 확인과 외부 백업 대상 선정(현재 외장 USB·다른 NAS·클라우드 백업 없음)

- 최초 PC 분실 시 운영자 복구 흐름
- 외부 접속 구현 방식(VPN 또는 Reverse Proxy)
- 품질 요구사항의 목표 수치
