# ADR-002: 일정 기반 가족 상태

- 상태: 승인
- 결정일: 2026-07-12

## 배경

가족의 현재 상황을 대시보드에서 파악하고 싶지만, 실시간 GPS는 모바일 앱의 백그라운드 권한, 배터리 사용량, 개인정보 보호 및 운영 복잡도를 크게 높인다. 딸은 위치를 전송할 전용 모바일도 없다.

## 결정

- 초기 버전은 GPS를 수집하지 않는다.
- 일정에 등록된 장소와 시작·종료 시간으로 가족의 예상 상태를 계산한다.
- 상태는 실제 위치가 아니라 `일정 기준`임을 명확하게 표시한다.
- 상태와 함께 해당 일정의 종료 시간을 표시한다.
- 실시간 GPS는 초기 버전 이후 별도 요구사항으로 재검토한다.

## 예시

```text
아빠 · 회사  일정 기준 · 18:00까지
엄마 · 일정 없음
딸   · 학원  일정 기준 · 17:00까지
```

## 결과

- 별도 네이티브 위치 추적 앱 없이 구현할 수 있다.
- 개인정보와 배터리 부담이 줄어든다.
- 일정이 실제 상황과 다를 수 있으므로 실제 위치처럼 단정해서는 안 된다.
- 일정에 장소와 대상 가족을 지정할 수 있어야 한다.
