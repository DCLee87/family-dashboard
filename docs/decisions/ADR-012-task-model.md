# ADR-012: 할 일 기본 구조

- 상태: 승인
- 결정일: 2026-07-12

## 배경

가족 할 일에는 한 사람이 담당하는 일과 여러 사람이 함께 수행하는 일이 있다. 일정과 유사한 담당 가족 구조를 사용하면 가족별 필터와 화면 표시를 일관되게 구성할 수 있다.

## 결정

- 할 일에는 제목과 담당 가족을 1명 이상 지정한다.
- 여러 가족을 하나의 할 일 담당자로 지정할 수 있다.
- 마감일과 마감 시각은 선택 사항이다.
- 우선순위는 `보통`과 `중요`를 지원하고 기본값은 `보통`이다.
- 선택적인 메모를 지원한다.
- 기본 상태는 `미완료`와 `완료`다.
- 여러 담당자가 있는 할 일은 관리자 모드에서 한 번 완료 처리하면 항목 전체를 완료로 변경한다.
- 완료 시각과 사용 기기를 기록하고 완료를 취소해 미완료로 되돌릴 수 있다.
- 딸은 보기 전용이므로 완료 상태를 변경할 수 없다.

## 결과

- 개인 및 가족 공동 할 일을 하나의 모델로 표현할 수 있다.
- 담당자별 개별 완료 진행률은 초기 구조에서 지원하지 않는다.
- 완료한 부모 개인은 공용 PIN만으로 식별할 수 없으므로 사용 기기와 시각만 확실하게 기록한다.
- 반복, 알림, 삭제·복구 및 완료 항목 보관 정책은 별도로 확정해야 한다.
