# ADR-018: 등록 기기의 안전한 외부 접속

- 상태: 원칙 승인, 구현 검증 필요
- 결정일: 2026-07-13

## 배경

부모는 집 밖에서도 모바일로 가족 정보를 조회하고 관리자 작업을 수행하고 싶다. NAS 서비스를 인터넷에 단순 공개하고 4자리 PIN만 사용하면 가족 데이터와 관리자 기능이 공격에 노출된다.

## 결정

- 등록된 부모 PC·모바일은 외부에서 HTTPS로 접속할 수 있다.
- 외부 미등록 기기에는 가족 보기 데이터를 제공하지 않는다.
- 외부 접속은 등록 기기 또는 사용자 세션 인증 후 보기 모드로 시작한다.
- 관리자 작업은 추가로 4자리 PIN을 입력해 관리자 모드를 잠금 해제한다.
- Push 알림 링크에도 동일한 외부 인증 규칙을 적용한다.
- 관리자는 등록 기기와 마지막 사용 시각을 확인하고 분실 기기의 권한과 Push 구독을 폐기할 수 있다.
- DSM 관리자 화면과 Container Manager는 대시보드 외부 접속을 위해 인터넷에 공개하지 않는다.
- VPN과 HTTPS Reverse Proxy 중 구체적인 구현은 Elaboration의 위협 모델과 실제 기기 검증 후 결정한다.
- 등록 기기는 관리자가 폐기하기 전까지 유지하며 인증정보는 사용자 조작 없이 안전하게 갱신한다.
- 90일 미사용 기기는 관리자에게 경고하지만 자동 폐기하지 않는다.

## 결과

- 외부 접속의 편의성과 가족 데이터 보호를 함께 요구한다.
- 기기 최초 등록, 자격정보 안전 저장, 만료, 갱신 및 폐기 기능이 필요하다.
- HTTPS 인증서, 도메인 또는 사설 네트워크 접근 방식이 필요하다.
- 직접 포트 공개 여부는 편의성만으로 결정하지 않고 위협 모델 검토를 거친다.
- 서버 복원 시 기기 인증정보를 복원하지 못하면 해당 기기를 다시 등록해야 한다.
