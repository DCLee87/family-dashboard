# ADR-015: 완료 할 일 보관과 삭제

- 상태: 승인
- 결정일: 2026-07-12

## 배경

완료한 할 일을 계속 기본 화면에 표시하면 미완료 항목을 찾기 어렵다. 그러나 반복 할 일의 수행 기록과 실수 복원을 위해 완료 이력을 즉시 삭제해서는 안 된다. 사용자가 명시적으로 삭제한 항목과 단순히 완료한 항목도 구분할 필요가 있다.

## 결정

- 완료한 할 일은 완료 당일 기본 화면에 취소선으로 표시한다.
- 완료 후 7일이 지나면 기본 화면에서 숨기고 완료 기록으로 이동한다.
- 완료 기록은 최소 1년 동안 보관하고 1년 이후에도 자동 영구 삭제하지 않는다.
- 관리자 모드에서 완료 기록을 조회하고 미완료로 복원할 수 있다.
- 사용자가 할 일을 직접 삭제하면 휴지통으로 이동하고 30일 동안 보관한다.
- 관리자 모드에서 휴지통 복원과 확인 후 영구 삭제를 제공한다.
- 휴지통에서 30일이 지나면 자동 영구 삭제한다.
- 반복 할 일 삭제 시 `이번 회차만`과 `전체 반복`을 구분한다.

## 결과

- 기본 화면은 미완료 항목에 집중하면서 수행 이력을 유지한다.
- 완료 기록은 저장 공간을 사용하므로 운영 중 정리 기능이 필요할 수 있다.
- 완료 취소와 휴지통 복원은 서로 다른 작업이며 원래 상태 복원 규칙을 구분해야 한다.
- 반복 회차 기록을 이용한 통계나 보상 기능을 나중에 확장할 수 있다.
