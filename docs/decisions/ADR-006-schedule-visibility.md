# ADR-006: 일정 공개 범위

- 상태: 승인
- 결정일: 2026-07-12

## 배경

거실 TV는 방문객에게 보일 수 있고, 부모의 병원 일정과 같은 정보는 딸이나 공용 화면에 공개하고 싶지 않을 수 있다. 동시에 일정 기반 예상 상태는 일정 정보와 연결되므로 상세 내용을 숨긴 상태에서도 일관된 표시 정책이 필요하다.

## 결정

일정은 다음 공개 범위 중 하나를 가진다.

1. `가족 공개`: 부모, 딸 및 공용 화면에 상세 표시
2. `TV 요약`: 부모와 딸에게 상세 표시하고 TV에는 `개인 일정`으로 요약
3. `부모 전용`: 부모에게만 상세 표시하고 딸과 공용 보기 화면에는 숨김

추가 원칙:

- 권한 없는 화면에는 일정 제목, 실제 장소 및 상세 내용을 전달하지 않는다.
- 부모 전용 일정이 예상 상태에 영향을 주면 실제 장소 대신 비식별 상태를 사용하거나 상태 자체를 숨긴다.
- 새 일정의 기본 공개 범위는 `가족 공개`다.
- 일정 등록·수정 화면에는 공개 범위를 눈에 띄게 표시하고 사용자가 변경할 수 있게 한다.
- 부모 전용 예상 상태의 구체적인 숨김 방식은 별도로 확정한다.

## 결과

- 거실 TV와 공용 기기에서 민감한 가족 정보를 보호할 수 있다.
- 조회 API가 사용자 및 기기 권한에 맞게 데이터를 필터링해야 한다.
- 화면에서만 내용을 감추는 방식은 사용할 수 없다.
- 일정 등록 화면에서 공개 범위를 이해하기 쉽게 선택할 수 있어야 한다.
- 민감한 일정은 사용자가 기본값을 `TV 요약` 또는 `부모 전용`으로 변경해야 한다.
