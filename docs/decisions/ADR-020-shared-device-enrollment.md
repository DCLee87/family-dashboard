# ADR-020: TV와 공용 태블릿의 로컬 등록

- 상태: 승인
- 결정일: 2026-07-13

## 배경

거실 TV는 리모컨으로 QR 코드를 스캔하기 어렵고 항상 보기 전용이어야 한다. 공용 태블릿은 평소 보기 모드이지만 부모가 PIN으로 관리자 모드를 사용할 수 있다. 두 기기는 부모 개인 모바일과 달리 집 밖에서 사용할 필요가 없다.

## 결정

### TV

- 미등록 TV는 집 안 네트워크에서 6자리 일회용 연결 코드를 표시한다.
- 코드는 10분 동안 유효하며 등록된 PC 관리자 화면에서 입력하고 승인한다.
- 승인된 TV에는 보기 전용 기기 프로필과 독립 인증정보를 발급한다.
- TV에서는 PIN 입력과 편집·관리 기능을 제공하지 않는다.
- TV 인증정보는 집 안 네트워크에서만 사용한다.

### 공용 태블릿

- 등록된 PC에서 10분 일회용 QR 코드와 대체 입력 코드를 생성한다.
- 집 안 네트워크에서 태블릿이 코드를 제출하고 PC에서 최종 승인한다.
- 승인된 태블릿에는 기본 보기 모드와 PIN 기반 관리자 모드를 제공한다.
- 공용 태블릿 인증정보는 집 안 네트워크에서만 사용한다.

### 공통

- TV와 태블릿은 각각 독립된 기기 이름과 인증정보를 가진다.
- 관리자 모드에서 각 기기를 개별 폐기할 수 있다.
- 등록 상태는 수동 폐기 전까지 유지하고 인증정보는 사용자 조작 없이 갱신한다.
- 90일 미사용 시 장기 미사용 경고를 표시하되 자동 폐기하지 않는다.

## 결과

- TV 리모컨 환경에서도 짧은 코드로 등록할 수 있다.
- TV 자격정보가 유출돼도 서버가 관리자 API 접근을 거부해야 한다.
- 로컬 전용 기기의 외부 접속을 서버에서 제한할 네트워크 판정 방식이 필요하다.
- 공용 태블릿을 나중에 외부에서도 사용하려면 별도의 정책 변경과 재등록이 필요하다.
