# ADR-021: 최초 신뢰 PC와 초기 설정 코드

- 상태: 승인
- 결정일: 2026-07-13

## 배경

부모 모바일, TV와 공용 태블릿 등록은 신뢰할 수 있는 PC의 승인을 전제로 한다. 하지만 설치 직후에는 아직 등록된 기기가 없으므로 안전한 최초 신뢰 기준점을 만들어야 한다.

## 결정

- 컨테이너 최초 실행 시 충분한 난수의 일회용 초기 설정 코드를 생성한다.
- 코드는 Container Manager에서 확인할 수 있는 보호된 초기 로그에 한 번 표시한다.
- 최초 설정 화면은 집 안 네트워크에서만 접근할 수 있다.
- 사용자는 초기 설정 코드를 입력하고 4자리 관리자 PIN을 설정한다.
- 서버는 PIN 복구 코드를 한 번 표시하고 사용자가 별도 보관했음을 확인하게 한다.
- 설정을 수행한 PC에 최초 신뢰 PC 프로필과 독립 인증정보를 발급한다.
- 등록 완료 즉시 초기 설정 코드를 무효화하고 최초 설정 화면을 닫는다.
- 최초 설정 완료 전에는 외부 접속, Push 등록 및 가족 데이터 기능을 활성화하지 않는다.

## 결과

- 공개된 기본 비밀번호 없이 최초 관리자 기기를 등록할 수 있다.
- Container Manager 로그를 볼 수 있는 NAS 운영자가 초기 설정 권한을 가진다.
- 초기 설정 로그와 코드가 일반 운영 로그에 장기간 남지 않도록 처리해야 한다.
- 최초 PC와 복구 코드를 모두 잃어버린 경우의 운영자 복구 절차가 별도로 필요하다.
