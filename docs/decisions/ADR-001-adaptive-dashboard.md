# ADR-001: 기기별 적응형 대시보드

- 상태: 승인
- 결정일: 2026-07-11

## 배경

TV는 많은 가족 정보를 한 화면에 보여줘야 하지만 모바일과 태블릿에서는 화면 크기와 사용 목적 때문에 핵심 정보만 보여줄 필요가 있다.

## 결정

- 단순 반응형 레이아웃에 더해 TV·PC·태블릿·모바일별 기기 프로필을 사용한다.
- 화면 크기에 따라 레이아웃을 자동 조정한다.
- 기기 프로필에 따라 표시하는 정보 자체를 다르게 구성한다.
- 관리자는 PC에서 기기별 위젯의 표시 여부와 순서를 변경할 수 있다.
- 초기 버전에서는 자유로운 드래그 배치를 지원하지 않는다.

## 결과

- 각 기기에서 목적에 맞는 화면을 제공할 수 있다.
- 기기별 화면 설정 데이터와 기본 프리셋이 필요하다.
- 작은 PC 창을 모바일로 오인하지 않도록 화면 크기와 기기 프로필을 함께 사용해야 한다.
