# ADR-019: 부모 모바일의 집 안 등록

- 상태: 승인
- 결정일: 2026-07-13

## 배경

외부 접속을 허용하려면 어떤 모바일이 부모의 신뢰할 수 있는 기기인지 안전하게 확인해야 한다. 공개된 로그인 화면에서 4자리 PIN만 입력해 기기를 등록하면 공격자가 가족 데이터에 접근할 수 있다.

## 결정

- 등록된 PC의 관리자 모드에서 부모 모바일 등록을 시작한다.
- 서버는 10분 동안 유효한 일회용 QR 코드와 대체 입력 코드를 발급한다.
- 최초 등록은 기본적으로 가족 시스템과 같은 집 안 네트워크에서만 허용한다.
- 모바일에서 소유자 `아빠` 또는 `엄마`와 기기 이름을 지정한다.
- 코드 제출 후 등록된 PC에서 요청을 최종 승인한다.
- 승인된 모바일에는 기기 전용 인증정보를 발급한다.
- 등록 코드는 한 번 사용하거나 만료·거절되면 다시 사용할 수 없다.
- 기기 교체 시 인증정보를 복사하지 않고 새 등록을 수행한다.
- 등록 기기는 관리자가 폐기하기 전까지 유지하고 기기 인증정보는 자동 갱신한다.
- 90일 동안 사용하지 않은 기기는 장기 미사용으로 표시하지만 자동 폐기하지 않는다.

## 결과

- 별도 부모 계정 로그인을 반복하지 않고도 신뢰할 수 있는 모바일을 구분할 수 있다.
- 소유자 속성으로 알림 수신자를 아빠·엄마 기기에 연결할 수 있다.
- PC가 등록 승인과 PIN 복구의 신뢰 기준점이 된다.
- 초기 PC 자체의 안전한 설정 및 분실 시 복구 절차가 중요해진다.
- QR 스캔을 지원하지 못할 때 사용할 대체 입력 코드가 필요하다.
