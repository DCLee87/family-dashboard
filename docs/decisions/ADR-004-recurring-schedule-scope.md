# ADR-004: 초기 반복 일정 범위

- 상태: 승인
- 결정일: 2026-07-12

## 배경

학교, 학원 및 정기 출근처럼 매주 반복되는 가족 일정이 많다. 반복을 지원하지 않으면 같은 일정을 계속 입력해야 한다. 반면 모든 달력 반복 규칙을 초기 버전에서 지원하면 구현과 예외 처리의 복잡도가 크게 증가한다.

## 결정

초기 버전은 다음 반복 기능을 지원한다.

- 반복하지 않는 일회성 일정
- 매주 특정 요일 반복
- 한 주에 여러 요일 선택
- 반복 시작일 필수
- 종료일 지정 또는 종료일 없는 반복
- 특정 회차 한 번만 취소
- 특정 회차의 시간, 장소 또는 제목만 변경

초기 버전에서 다음은 지원하지 않는다.

- 매월 특정 날짜
- 매월 N번째 요일
- N주 또는 N개월 간격
- 복잡한 사용자 정의 반복 규칙

## 예시

```text
피아노 학원
대상: 딸
반복: 매주 화·목
시간: 15:00~17:00
기간: 2026-03-02~2026-07-31
장소: 피아노 학원
```

## 결과

- 가족의 주요 정기 일정을 반복 입력 없이 관리할 수 있다.
- 반복 원본과 특정 회차의 취소·변경 예외를 함께 저장해야 한다.
- 일정 조회와 상태 계산은 원본 규칙에 예외를 적용한 최종 회차를 사용해야 한다.
- 복잡한 월간 규칙은 실제 필요성이 확인될 때 후속 기능으로 추가한다.
