# ADR-013: 반복 할 일과 회차별 완료

- 상태: 승인
- 결정일: 2026-07-12

## 배경

양치, 분리수거, 연습처럼 반복되는 할 일은 하나의 완료 상태만 사용하면 다음 회차를 표현할 수 없다. 각 날짜의 수행 여부를 독립적으로 기록해야 한다.

## 결정

- 반복 없음, 매일 반복, 매주 선택 요일 반복을 지원한다.
- 반복 시작일은 필수이며 종료일은 선택 사항이다.
- 각 반복 회차는 독립적인 미완료·완료 상태와 완료 시각을 가진다.
- 현재 회차를 완료해도 다음 회차는 새 미완료 항목으로 제공한다.
- 특정 회차만 건너뛸 수 있다.
- 월간, N일 간격 및 복잡한 사용자 정의 반복은 초기 범위에서 제외한다.

## 예시

```text
피아노 연습
담당: 딸
반복: 매주 월·수·금

월요일 회차: 완료
수요일 회차: 미완료
금요일 회차: 아직 시작 전
```

## 결과

- 반복 원본과 회차별 완료·건너뛰기 기록을 별도로 저장해야 한다.
- 완료 통계나 보상 기능은 회차 기록을 기반으로 나중에 확장할 수 있다.
- 회차 생성 시점과 누락된 과거 회차의 처리 방식은 Elaboration에서 확정한다.
