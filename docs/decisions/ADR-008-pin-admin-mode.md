# ADR-008: PIN 기반 관리자 모드

- 상태: 승인
- 결정일: 2026-07-12

## 배경

관리자 기능을 PC로만 제한하면 부모가 모바일이나 공용 태블릿에서 일정 삭제·복원 같은 작업을 하기 불편하다. 가족 규모가 작으므로 사용자별 관리자 역할 선택보다 공용 PIN으로 관리자 기능을 잠금 해제하는 방식이 단순하다.

## 결정

- PC, 부모 모바일 및 공용 태블릿은 기본적으로 보기 모드로 동작한다.
- 공용 관리자 PIN을 입력하면 관리자 모드를 잠금 해제한다.
- 관리자 모드에서는 일정·할 일·공지 관리와 일정 삭제·복원·영구 삭제를 수행할 수 있다.
- 일정 시간 조작이 없으면 관리자 모드를 자동 종료한다.
- 거실 TV는 항상 보기 전용이며 PIN 입력 화면과 관리자 기능을 제공하지 않는다.
- 외부 접속에서는 신뢰할 수 있는 기기 또는 인증된 세션을 먼저 요구하고 PIN은 권한 승격에만 사용한다.
- 변경 이력에는 최소한 사용 기기와 변경 시각을 기록한다.
- 관리자 PIN은 4자리 숫자로 한다.
- 마지막 조작 후 10분이 지나면 관리자 모드를 자동 잠금한다.
- PIN을 5회 연속 잘못 입력하면 해당 기기에서 5분 동안 입력을 차단한다.
- 화면에 관리자 모드 여부와 즉시 잠금 버튼을 제공한다.
- 초기 설정에서 PIN 복구 코드를 한 번 표시한다.
- PIN 복구는 등록된 PC에서만 제공하며 모바일, 태블릿 및 TV에서는 제공하지 않는다.
- 복구 코드 원문은 저장하지 않고 사용 후 즉시 무효화하며 새 복구 코드를 발급한다.

## 결과

- 부모가 PC에 가지 않고 모바일과 태블릿에서도 관리 작업을 수행할 수 있다.
- 공용 PIN만으로 부모 개인을 구분할 수 없으므로 변경자를 개인 이름으로 정확히 기록하지 못한다.
- PIN 유출과 무차별 입력에 대비해 실패 제한, 안전한 저장 및 자동 잠금이 필요하다.
- 4자리 PIN은 가능한 조합이 적으므로 실패 제한을 화면이 아닌 서버에서 강제해야 한다.
- 계정, 백업, 외부 접속 등 고위험 설정에 추가 확인이나 PC 제한이 필요한지는 Elaboration에서 검토한다.
- 등록된 PC를 잃어버린 경우의 운영자 복구 절차는 Elaboration에서 별도로 설계해야 한다.
