# ADR-010: 적응형 PWA와 NAS 단일 컨테이너

- 상태: 아키텍처 후보 승인
- 결정일: 2026-07-12

## 배경

PC, TV, iOS, Android 및 태블릿을 지원해야 하지만 DS216+II는 1 GB 메모리 제약이 있다. 기기별 네이티브 앱과 여러 서버 컨테이너를 별도로 운영하면 개발·배포·운영 비용이 커진다.

## 결정

- 모든 기기는 하나의 적응형 PWA를 사용한다.
- 기기 프로필과 화면 크기로 표시 정보와 레이아웃을 조정한다.
- 시놀로지 NAS에는 웹 화면, API 및 예약 작업을 포함한 단일 경량 애플리케이션 컨테이너를 우선 배포한다.
- 데이터 저장소는 SQLite를 우선 검증한다.
- 영속 데이터는 컨테이너 외부 NAS 볼륨에 저장한다.
- 네이티브 iOS·Android 앱과 앱스토어 배포는 초기 범위에서 제외한다.

## 결과

- 하나의 코드베이스와 배포본으로 주요 기기를 지원할 수 있다.
- iOS·Android의 PWA 및 Push 차이는 실제 기기에서 검증해야 한다.
- TV 브라우저 호환성이 부족하면 외부 TV 출력 장치를 대안으로 검토한다.
- 이 결정은 Elaboration의 실행 가능한 아키텍처 검증을 통과한 후 기준선으로 확정한다.
