# 기기와 권한

## 권한 매트릭스

| 기기 | 사용자 | 권한 및 목적 |
|---|---|---|
| PC | 본인, 배우자 | 기본 보기, 관리자 PIN 입력 후 관리자 모드 |
| 거실 TV | 가족 공용 | 항상 보기 전용 대시보드 |
| 가족 공용 태블릿 | 부모, 딸 | 기본 보기, 관리자 PIN 입력 후 관리자 모드 |
| 본인 모바일 | 본인 | 기본 보기, 관리자 PIN 입력 후 관리자 모드 |
| 배우자 모바일 | 배우자 | 기본 보기, 관리자 PIN 입력 후 관리자 모드 |

## 관리자 모드의 범위

- 일정 추가 및 수정
- 할 일 추가, 수정 및 완료 처리
- 공지·메모 작성
- 일정 삭제, 휴지통 복원 및 영구 삭제
- 기기별 화면 구성 변경
- 계정, 보안, 백업 및 서버 운영처럼 고위험인 시스템 설정은 기능별 추가 확인 또는 PC 제한을 추후 검토

## 관리자 PIN 정책

- PC, 부모 모바일 및 공용 태블릿의 기본 상태는 보기 모드다.
- 공용 관리자 PIN을 입력하면 관리자 모드가 활성화된다.
- 일정 시간 조작이 없으면 보기 모드로 자동 복귀한다.
- 거실 TV는 PIN 입력과 관리자 모드를 제공하지 않는다.
- 공용 태블릿에서 발생한 변경은 `공용 관리자`로 기록한다.
- PIN 원문은 저장하지 않는다.
- 관리자 PIN은 4자리 숫자다.
- 마지막 조작 후 10분이 지나면 관리자 모드를 자동 잠금한다.
- PIN을 5회 연속 잘못 입력하면 해당 기기에서 5분 동안 PIN 입력을 차단한다.
- 관리자 모드 여부를 화면에 명확히 표시하고 사용자가 즉시 잠글 수 있는 버튼을 제공한다.
- 초기 설정에서 PIN 분실에 대비한 복구 코드를 발급한다.
- PIN 복구는 등록된 PC의 복구 화면에서만 제공하고 모바일, 태블릿 및 TV에서는 제공하지 않는다.
- 복구 코드 원문은 서버에 저장하지 않으며 사용 후 기존 코드를 무효화하고 새 코드를 발급한다.

## 기기·세션 인증

- PIN은 관리자 권한을 잠금 해제하는 수단이며 외부 접속 사용자의 신원을 확인하는 유일한 로그인 수단으로 사용하지 않는다.
- 부모 개인 모바일과 PC는 신뢰할 수 있는 기기 또는 인증된 세션을 먼저 구성하고 그 안에서 PIN으로 관리자 모드를 잠금 해제한다.
- 구체적인 기기 등록과 로그인 방식은 Elaboration에서 확정한다.

## 일정 공개 범위

| 공개 범위 | 부모 개인 화면 | 딸 화면 | 거실 TV | 공용 태블릿 보기 상태 |
|---|---|---|---|---|
| 가족 공개 | 상세 표시 | 상세 표시 | 상세 표시 | 상세 표시 |
| TV 요약 | 상세 표시 | 상세 표시 | `개인 일정`으로 요약 | 상세 표시 |
| 부모 전용 | 상세 표시 | 숨김 | 숨김 또는 비식별 상태 | 숨김 |

- 부모 전용 일정이 예상 상태에 영향을 주면 보기 권한이 없는 화면에는 `개인 일정 · 종료 시각`으로 비식별 표시한다.
- 비식별 표시에는 일정 제목, 실제 장소, 메모 및 참여자 상세를 포함하지 않는다.
- 새 일정의 기본 공개 범위는 `가족 공개`다.
- 일정 등록·수정 화면에는 현재 공개 범위를 명확하게 표시하고 `TV 요약` 또는 `부모 전용`으로 변경할 수 있어야 한다.
