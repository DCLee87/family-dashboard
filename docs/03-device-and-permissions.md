# 기기와 권한

## 권한 매트릭스

| 기기 | 사용자 | 권한 및 목적 |
|---|---|---|
| PC | 본인, 배우자 | 전체 관리자 기능과 상세 편집 |
| 거실 TV | 가족 공용 | 보기 전용 대시보드 |
| 가족 공용 태블릿 | 부모, 딸 | 기본 보기 전용, PIN 인증 후 일부 편집 |
| 본인 모바일 | 본인 | 일부 편집과 조회 |
| 배우자 모바일 | 배우자 | 일부 편집과 조회 |

## 일부 편집의 범위

- 일정 추가 및 수정
- 할 일 추가, 수정 및 완료 처리
- 공지·메모 작성
- 사용자, 권한, 화면 구성 및 시스템 설정은 변경할 수 없음

## 공용 태블릿 PIN 정책

- 기본 상태는 보기 전용이다.
- 공용 편집 PIN을 입력하면 일부 편집 기능이 활성화된다.
- 일정 시간 조작이 없으면 보기 전용으로 자동 복귀한다.
- 공용 태블릿에서 발생한 변경은 `공용 편집 사용자`로 기록한다.
- PIN 원문은 저장하지 않는다.
- PIN 길이, 잠금 유지 시간 및 실패 제한은 추후 확정한다.

## 모바일 인증

- 부모 개인 모바일은 최초 인증 후 로그인 상태를 유지하는 방향을 우선 검토한다.
- 구체적인 인증 방식은 추후 결정한다.

## 일정 공개 범위

| 공개 범위 | 부모 개인 화면 | 딸 화면 | 거실 TV | 공용 태블릿 보기 상태 |
|---|---|---|---|---|
| 가족 공개 | 상세 표시 | 상세 표시 | 상세 표시 | 상세 표시 |
| TV 요약 | 상세 표시 | 상세 표시 | `개인 일정`으로 요약 | 상세 표시 |
| 부모 전용 | 상세 표시 | 숨김 | 숨김 또는 비식별 상태 | 숨김 |

- 부모 전용 일정이 예상 상태에 영향을 주면 보기 권한이 없는 화면에는 `개인 일정 · 종료 시각`으로 비식별 표시한다.
- 비식별 표시에는 일정 제목, 실제 장소, 메모 및 참여자 상세를 포함하지 않는다.
- 새 일정의 기본 공개 범위는 `가족 공개`다.
- 일정 등록·수정 화면에는 현재 공개 범위를 명확하게 표시하고 `TV 요약` 또는 `부모 전용`으로 변경할 수 있어야 한다.
